## 🚀 Built With
* **React.js** (Dynamic component rendering)
* **Tailwind CSS** (Modern, responsive styling)
* **Framer Motion** (Smooth animations)

## 📂 Project Structure
This portfolio is built to be easily maintainable. All content (projects, bio, certificates) is separated from the logic.
* `js/data.js` - Contains all text, links, and project data.
* `js/main.js` - The React application logic.
* `css/style.css` - Custom styling overrides.
